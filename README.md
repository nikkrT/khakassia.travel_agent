# 🤖 Туристический ИИ-агент ТИЦ Хакасия

**Туристический ИИ-агент** — это интеллектуальный виртуальный помощник, созданный для Туристического информационного центра (ТИЦ) Республики Хакасия. Приложение работает на базе **GigaChat** и помогает туристам планировать путешествия: подбирать туры, искать варианты размещения и получать рекомендации по отдыху на природе и праздничным мероприятиям.

Проект построен с использованием графовой архитектуры **LangGraph** для управления сложными диалоговыми сценариями и **Streamlit** для удобного пользовательского веб-интерфейса.

## 🌟 Возможности

- **Умный диалог**: Естественное общение на русском языке благодаря LLM GigaChat.
- **Управление контекстом (Memory)**: Агент помнит историю сообщений и контекст текущей беседы.
- **Сценарный роутинг**: Использование LangGraph позволяет гибко направлять запросы пользователя по нужным веткам (поиск жилья, подбор туров, советы по природе).
- **Интерактивный UI**: Удобный чат-интерфейс на базе Streamlit с кастомным оформлением и аватарами.
- **Сброс контекста**: Кнопка "Новый диалог" для быстрой очистки памяти и начала новой сессии.

## 🛠 Стек технологий

- **Язык**: Python 3
- **Интерфейс**: [Streamlit](https://streamlit.io/)
- **LLM / ИИ**: [GigaChat](https://developers.sber.ru/portal/products/gigachat)
- **Оркестрация**: [LangChain](https://www.langchain.com/) / [LangGraph](https://langchain-ai.github.io/langgraph/)

## 📂 Структура проекта

```text
├── app.py               # Главный файл запуска Streamlit-приложения
├── graph/               # Логика сборки графа состояний (LangGraph)
├── nodes/               # Узлы графа (отдельные шаги обработки запроса)
├── the_state/           # Описание структуры состояния (State) графа
├── tools/               # Инструменты агента (например, работа с памятью/БД)
├── promts/              # Промпты для LLM GigaChat
├── data/                # Данные для работы агента (базы туров, тексты)
├── images/              # Графические ресурсы (аватары, фоны)
└── requirements.txt     # Зависимости Python
```

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/nikkrT/khakassia.travel_agent.git
cd khakassia.travel_agent
```

### 2. Создание виртуального окружения
Рекомендуется использовать виртуальное окружение для установки зависимостей:
```bash
python -m venv venv
# Активация для Windows:
venv\Scripts\activate
# Активация для Linux/macOS:
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
Для работы приложения потребуется доступ к API GigaChat. 
Создайте файл `.env` в корневой директории и добавьте туда ваши ключи доступа (или настройте их в соответствии с логикой вашего приложения):
```env
GIGACHAT_CREDENTIALS="ваш_ключ_авторизации_gigachat"
```

### 5. Запуск веб-приложения
Запустите интерфейс Streamlit командой:
```bash
streamlit run app.py
```
После запуска откроется вкладка в браузере (по умолчанию `http://localhost:8501`), где вы сможете взаимодействовать с ИИ-агентом.

## 📝 Логирование
